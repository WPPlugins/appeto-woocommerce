== Description ==

Appeto is a powerful Persian online application generator for android and ios.
With this plugin you can connect your Woocommerce shop to your application that have been generated on appeto panel.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/appeto-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Learn more about how to connect plugin to appeto panel: <a href="http://appeto.ir/support/faqs.php?scvf=f7177163c833dff4b38fc8d2872f1ec6" target="_blank">http://appeto.ir/support/faqs.php?scvf=f7177163c833dff4b38fc8d2872f1ec6</a>
